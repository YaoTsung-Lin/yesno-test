
# YES / NO 即時投票系統（完整多人同步版）

===================================

【功能】

✅ 多人手機同步投票
✅ 即時更新票數
✅ 主持人控制下一題
✅ 8題 YES / NO
✅ 手機、電腦皆可使用
✅ 可直接部署上線

===================================

【本系統技術】

- Node.js
- Express
- Socket.io

===================================

【本機執行】

1. 安裝 Node.js
https://nodejs.org/

2. 開啟命令列

3. 執行：

npm install

4. 啟動：

npm start

5. 開啟：

http://localhost:3000

===================================

【免費部署（推薦 Render）】

1. 註冊 Render
https://render.com/

2. 建立 Web Service

3. 上傳本專案

4. Build Command：

npm install

5. Start Command：

npm start

6. 完成後即可取得網址

===================================

【專案結構】

server.js
package.json
public/index.html

===================================

【注意】

目前任何人都可按「下一題」。

若需要：
- 主持人權限
- QR Code
